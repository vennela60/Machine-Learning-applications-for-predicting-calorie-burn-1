Title:Calorie Burn Prediction Using Machine Learning

A Machine Learning web application that predicts the number of calories burned during physical activities using health and workout data.

# Overview
This project uses Machine Learning algorithms to estimate calories burned based on user inputs such as:
- Age
- Gender
- Height
- Weight
- Duration
- Heart Rate
- Body Temperature

The model was trained on fitness-related data and deployed using Flask.

# Algorithms Used
- XGBoost Regressor ✅
- Linear Regression
- Decision Tree
- Support Vector Regression

### Best Model
- **XGBoost Regressor**
- Accuracy: **99.88%**

| Heart Rate | Average heart rate |
| Body Temp | Body temperature |

# How to Run
## Clone Repository
```bash
git clone https://github.com/your-username/calorie-burn-prediction.git
```

## Install Requirements
```bash
pip install -r requirements.txt
```

## Run Application
```bash
python app.py
```

Open browser:
text:
http://127.0.0.1:5000/
